Tracking the desktop applications in real time and time spent on each application.


## Setup instructions:

- 1.  Clone this repository.


- 2.  cd into the repository.

            cd auto_timer

## Install and Run:

- 1.  Run requirements file

            sudo pip install -r requirements.py

- 2.  Run

            python autotimer.py

## Dependencies:
Mac Depencies
- selenium

Windows Depencies

- pywin32
- python-dateutil
- uiautomation
